# Python CLI Calculator
A simple **command-line 
calculator** written in Python 
that supports the basic arithmetic 
operations: - Addition (`+`) - 
Subtraction (`-`) - Multiplication 
(`*`) - Division (`/`) The project 
follows **clean code principles**, 
separating **calculation logic** 
from the **CLI (user interface)**. 
---
## Features
- Command-line interface (CLI) - 
Supports `+`, `-`, `*`, `/` - 
Clean and readable code - Business 
logic separated from user 
input/output - Easy to extend with 
more operations ---
